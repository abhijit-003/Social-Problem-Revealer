<h2>Social Problem Revealer</h2>

<p>As a name This project is made for the social problems.
The idea behind this project is This system is creating bridege between muncipality and the peoples where people can upload their problems which displayed on muncipality dashboard(admin side). Admin can change the Status of the problem according to the updation in the work of problems.<p>

<h4>Why This Project</h4>
<p>now days people face lots of social problems like road, light,polution etc, but the process of registering the complaints againsts these issues is to lengthy and complicated. To save the the time and avoid complexity we made this system.<p>

<h4>Features</h4>
<p><b>-</b>User can add social problems</p>
<p><b>-</b>Different Problem categories</p>
<p><b>-</b>Problem Status</p>
<p><b>-</b>Manage Profile</p>
<p><b>-</b>View Problems</p>

<p><b>-</b>Change Status</p>
